# Data Science @ UC Berkeley

## Projects
**Housing Price Predictor** 🏠

Built a linear regression model to predict sales prices for properties of Cook County, Illinois. Explored Cook County Assesor's Office dataset through exploratory visualizations and engineered useful features for prediction using Python. Evaluated and fitted model by computing the root mean squared error of 10 K-cross folds. 

An interesting project and case study that nudges you to consider the human context and the social implications of the predicted home value. Challenges you to question: What does fairness means for an algorithm? What is the difference between fair and accurate? How does an inaccurate prediction effect people of different identities and positionalities?


**Spam Email Detector** 📧

Built a binary classifier with logistic regression to classify emails as spam/non-spam using features from text. Identified key features to distinguish emails through exploratory data analysis.
